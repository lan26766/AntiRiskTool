使用操作手册

系统要求：Windows 10 / 11（64位），必须以管理员身份运行，.NET 8.0运行时。

安装：解压后右键 AntiRiskTool.exe → 以管理员身份运行 → 输入激活码 → 自动环境检测。

命令行模式：支持 protect（一键防护）、restore（恢复）、random（随机生成）、batch（批量生成）、read（读取硬件）、force-restore（强制还原）等参数。

点击"一键启动隐私防护模式"，自动执行：清理浏览器痕迹 → 重置系统身份 → 调整设备标识 → 环境安全检测。失败自动回滚。

自动化挂机：设置间隔、勾选执行项、点击"启动自动防护"，即可自动循环运行。

9张硬件卡片，左=原始值，右=伪装值。操作流程：重新读取 → 智能生成 → 应用 → 保存方案。

支持方案管理面板：导入/导出/批量生成/一键应用。

导入IP：粘贴 IP:端口 或 socks5://IP:端口:用户名:密码 格式批量导入。

检测：点"批量检测代理IP"并发测速，状态分为全功能正常/基础连通/认证失效/失效。

启动：确保有可用节点 → 点"启用代理" → 查看状态面板确认。

⚠️ 断网保护：代理断开后网络自动锁定，这是防泄露真实IP的安全机制。检查代理 → 切换新IP → 解除锁定。

用于母机多开模拟器，每窗口独立IP。注意：使用此模块前会自动停止全局代理（二者互斥）。

步骤：① 导入IP并检测 → ② 启动模拟器 → ③ 刷新进程列表 → ④ 勾选进程+IP → 绑定 → ⑤ 启动代理。

支持一键批量分配、一键批量重绑、替换失效IP等操作。

三组清理项：浏览器痕迹（Chrome/Edge/Firefox缓存/Cookie/历史/密码）、系统日志（临时文件/事件日志/注册表/DNS/Prefetch）、应用缓存（微信QQ/模拟器/游戏平台）。

操作流程：展开分组 → 勾选 → 扫描 → 执行选中。
